# Youtube for Gamepad
This app runs the TV/Console UI for youtube on pc allowing for controller inputs. Currently it has 100% support for xbox controllers I am unsure about Duelshock/DuelSense though support is intended to be added at some point. This app was just made because I couldn't find anything like it and its very much not good. But it works for the most part and ill maintain it for a bit.

## Controller Controls:
* A = Select
* B = Back
* Dpad and Leftstick = Directions
* LB = Skip Back
* RB = Skip Forward
* Start = Pause/Resume

## Keyboard Controls
> Keyboard is automatically supported by the UI though it wasnt my intended use.

> Note that when search you cant really use the keyboard as some keys are bound to random things that I will not be listing
* Enter = Select
* Esc = Back
* Arrow Keys = Directions
* Space = Pause
